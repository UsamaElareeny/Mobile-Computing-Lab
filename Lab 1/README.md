# Mobile Computing Lab 1

## Assignment 01 Report Notes

### Fonts Used

- `Poppins_700Bold` (screen title)
- `Poppins_500Medium` (goal item text)
- `Poppins_400Regular` (input text)

Fonts are loaded globally in `app/_layout.tsx` using `@expo-google-fonts/poppins` and `expo-font`.

### Color Palette Used

- Primary: `#4E79F9`
- Background: `#F7F9FC`
- Surface: `#FFFFFF`
- Border: `#CBD5E1`
- Main text: `#1F2937`
- Placeholder text: `#999`
- Completed item background: `#E2E8F0`
- Completed item text: `#94A3B8`
